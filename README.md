# OpenArvello

OpenArvello is a project aiming to keep Arvello fully open source after the controversial relicensing to Creative Commons.

The latest branch of official Arvello is no longer public.

## Updates

OpenArvello will try to achieve feature parity with Arvello without using any new code from it.
Security updates will be shipped regardless of when Arvello updates.
